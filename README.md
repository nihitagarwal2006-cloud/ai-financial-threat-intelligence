## What is FinSentinel?

FinSentinel is a free AI-powered tool that helps people in India detect financial scams instantly. Paste any suspicious message, URL, or screenshot and get a full risk analysis in seconds — no keyword matching, real AI.

---

## Features

| Feature | Description |
|---------|-------------|
| 💬 **Message Analyzer** | Paste any suspicious SMS, WhatsApp or email |
| 📞 **Call Transcript Analyzer** | Detect vishing & phone scams |
| 🔗 **URL Scanner** | Check if a link is a phishing site |
| 🖼️ **Screenshot Analyzer** | Upload a screenshot, AI reads it |
| 💳 **UPI Fraud Checker** | Check if a UPI ID looks fraudulent |
| 🗺️ **India Scam Heatmap** | Live state-wise scam report map |
| 📊 **Live Scam Tracker** | Community-powered scam type & platform tracker |
| 🏦 **Bank Helpline Directory** | One-click fraud helplines for 16+ banks |
| 🚨 **RBI & SEBI Fraud Alerts** | Official government fraud warnings |
| 🌍 **Multi-language Support** | English, Hindi, Tamil, Telugu, Bengali, Marathi |
| 🌙 **Dark / Light Mode** | Toggle anytime |
| 📞 **1930 Cybercrime Helpline** | Quick-dial button always visible |

---

## How it Works

1. **Paste or upload** any suspicious message, URL or screenshot
2. **AI analyzes** full context — not just keywords
3. **Get a report** with risk score, red flags & safety tips
4. **Stay safe** — block, report or call 1930
